Service Workers

Service Worker - Get Along

The purpose of this exersice is to have an idea how service workers work. How we can use it for offline experience. At granular level we will se How we can control the overall process.

Information
Objectives:

  • To understand how service workers work. How we can control static and dynamic content.
  • Research on performance with or without service workers.
  • Use newly available replacement of XHR that is fetch() Interface.
  • Bootstrap-4, Newly available standard. Get used to newly available components.
  • Flex-box, Newly available standard, Get used to it and see how it can be used with bootstrap.
